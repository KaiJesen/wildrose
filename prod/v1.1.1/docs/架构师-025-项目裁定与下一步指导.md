**关键观察**：

1. **A3a 慢涨**：valid 9 组网格中仅有 τ=0.35~0.55 + edge=-0.15 这 3 组勉强可用（valid return 0.38%，覆盖 21.9%）；test 上只开 2 笔 slow_up，PnL −1.21%。结论：**c1_pw20 的 participate_score 在慢涨上下文中无法选到正收益开仓**。

2. **A3b 标准趋势混合**：valid 6 组网格**全部失败**（return −3.41%，覆盖 15.6% < B0 的 18.75%），tune_summary 明确 `"best": null`。test 上默认 mix 靠 2 笔 `OPEN_LONG_SIGNAL` 将 coverage 推至 30%，但回归年化仅 6.81%。**结构与 024 的 w_part≥0.37 膝点完全同构**（coverage↑ return↓，斜率更陡）。

3. **已形成覆盖面上限**：在 c1_pw20 这个固定模型 checkpoint 上，无论 TEQ 调 w_part（024）还是多通道扩展（025），Pareto 前沿无法同时达到 return≥8.84% + coverage≥28%。

## 3. 选项裁定

### 3.1 选项 A：接受「coverage-only」探索线 — **驳回**

| 理由 | 说明 |
|------|------|
| **违背 024/025 双轨契约** | 探索线 return≥8.84%（95% v022）是自 023 复审起跨版本一致的底线，A3b 仅 6.81% 差距过大 |
| **teq_pnl 已转负** | B0 teq_pnl=+0.80% → A3b teq_pnl=−0.23%；新开仓为负收益交易 |
| **与 023 phase3 同构** | 023 phase3「标准开仓放宽」test return −2.03%，A3b 的 −2.20pp 是同一模式的重现 |
| **边际收益递减** | 多 3.3pp coverage 少 2.20pp return，若再推覆盖至 35% 预计 return 趋近于 0 或负 |

**裁定**：不通过 coverage-only 探索线。`return` 和 `coverage` 是双轨，不可单边验收。

---

### 3.2 选项 B：试 A3c Floor（Participation Floor） — **跳过，不实验**

| 理由 | 说明 |
|------|------|
| **信号工具相同** | A3c 复用 c1_pw20 的 `participate_score`，A3a/A3b 已证明该分数在慢涨/标准通道中无法提供正期望分离 |
| **约束更弱** | A3c 仅要求 `confirmed leg + flat + participate_score≥τ_floor` → 最小 probe，比 A3a（slow_up 参与门）和 A3b（std_trend_edge 混合）更弱 |
| **历史佐证** | 023 phase1a（无门慢涨，仅 0062e）test return 1.99%；A3a 加了参与门后仍仅 7.25%；A3c 门比 A3a 更宽、质量更差 |
| **预计结果可推断** | valid 上 A3a 最松的 τ=0.35 已回吐至 0.38% return；A3c 的 τ_floor 即使取 0.55，重复 A3a 的失败模式概率 >95% |

**裁定**：A3c 无需再实验。架构师直接驳回该选项，节省 1 周工程时间。若项目经理坚持需要实验验证，可单开 1 个最小 PR（仅 config + 1 次 test），但预测结果已明确。

## 6. 下一步建议（026 方向）

### 6.1 根因回溯

```
c1_pw20 的 participate_score 在慢涨/标准通道无区分力
  → ideal_participate 标签是 hindsight，模型只学到了 5-bar cum 特征
  → 腿内参与需要更长视野的上下文特征（12~48 bar）
  → 但 024 的多视野头（return_pred_12/24/48）在 constrained 训练中退化 <5% cum_IC
    → 表明被 0062e 基线冻结约束限制了腿标签的表达空间
```

### 6.2 推荐方向（按优先级排序）

#### 方向 1：标签升级（026a）— 最高优先

现有的 `ideal_participate_*` 标签是 **确定性 hindsight**（ROI>0 且 MAE 可控），不包含「可参与但风险较高」的中间态。建议：

```text
标签 v2：
  ideal_participate = 0 / 1 / 2（强参与 / 弱参与 / 不参与）
  弱参与 = ROI>0 但 MAE 在 [1.5, 2.5] ATR，或 progress_ratio >0.35
  损失改用 ordered BCE 或 triplet margin
```

同时取消 024 的 "退化 ≤5% cum_IC" 约束，允许合理 trade-off。

#### 方向 2：约束训练权重重新调参（026b）

024 的 `cum_IC 退化 ≤5%` 门可能过紧。放宽至 ≤10% 后重跑 ablation 链，观察 `participation_auc` 能否突破 0.60。

#### 方向 3：多视野独立 trunk（026c）

当前多视野头与 5-bar head 共享 trunk。若方向 2 仍无效，考虑 `return_pred_24` 使用独立 trunk 分支（与 0062e 解耦），但训练量 +50%。

### 6.3 不建议的方向

| 方向 | 原因 |
|------|------|
| 重训 0065a 更大参数量 | c1 系列已试三个参数量，未看到突破 |
| 加入外部特征（order book / macro） | 缺乏接入管线，且需要重新冻结回测数据 |
| 降低探索线 coverage 至 26% | 025 立项时已明确排除「选项 C」 |
