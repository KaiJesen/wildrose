## 2. 实验目的

验证 0062e 新结构分支训练方案在 **不同随机种子** 下是否可重复，排除单次 seed=42 偶然达标的可能。

项目经理关注点（009 §4 阶段一）：

1. `cum_return_ic`、`macro_f1` 跨 seed 方差是否可控  
2. direction/risk **坍缩 gates** 是否始终通过  
3. 是否每轮均可产出 **eligible checkpoint**（非 diagnostic_last）

---

## 6. 稳定性自检（训练师口径）

对照 `document/008/项目进度纪要.md` §5.2 建议与 009 阶段一目标：

| 检查项 | 结果 |
|--------|------|
| 三轮 decision 均为 accept | ✅ |
| 三轮测试坍缩 gates 全通过 | ✅ |
| 三轮均有 eligible checkpoint | ✅ |
| 三轮 cum_return_ic ≥ 0.08 | ✅（最低 0.113） |
| 三轮 macro_f1 ≥ 0.32 | ✅（最低 0.323） |
| 三轮 return_ic > 0 | ✅ |
| cum_return_ic 跨 seed std ≤ 0.02 | ✅（0.010） |
| macro_f1 跨 seed std ≤ 0.02 | ✅（0.005） |

**训练师结论**：新结构分支（0062e 方案）在 seed 42/43/44 上 **表现稳定**，非单次偶然；direction/risk 坍缩未复现；核心指标 `cum_return_ic` 与 `macro_f1` 方差小。

**保留关注**：step `return_ic` 仍稳定在 0.005–0.009，与 008 已知 trade-off 一致，不在本阶段解决范围内。
