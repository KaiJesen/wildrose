## 2. 总体结论

025 的立项动因 **成立**，方向 **正确**，是 024 选项 D 的自然延伸。024 已在冻结 phase1c + Scheme B（仅 TEQ 消费 `teq_edge_*`）下完成工程闭环，并在 test 上形成可复现的 **Pareto 膝点**：

| 指标 | A0（0062e + phase1c） | A2 best（c1_pw20 + TEQ wp=0.35） | 探索线门槛 |
|------|----------------------|----------------------------------|------------|
| `total_return` | +7.77% | **+9.01%** ✅ | ≥ 8.84% |
| `leg_count_coverage_ratio` | 16.7% | **26.7%** ❌（−1.3pp） | ≥ 28% |
| `teq_open` | 1 | 3 | — |

继续在同一消费面上调 `w_part`：**16/16** 配置无法双门 PASS；`w_part≥0.37` 时 coverage 可达 30%，但 return 跌至 8.55%（−0.29pp）。**「规则消费面过窄」** 的归因有充分实验支撑，025 将参与分数扩展至慢涨 / 标准趋势白名单通道，在业务逻辑上 **必要且克制**（相对 024 §6.4 全局 effective_edge 方案 A）。

但 v1.0 立项材料仍存在 **三类会直接影响实验归因与投产判断** 的问题：

1. **慢涨通道有 023 强负面先例**（phase1a：return 9.31%→1.99%，coverage 仅升至 26.67%），A3a 的「参与门」是否足以隔离历史失败模式，方案未给出定量先验。
2. **探索线闭合（28%）与投产线（35%）差距仍大（7pp）**，025 成功仅闭合探索线时，Phase 4 大概率仍 FAIL——立项需明示「025 主目标是探索线，非 prod 提名」。
3. **A3b 标准趋势条件混合** 与 023 phase3「标准开仓放宽」高度同构，虽有 confirmed-leg 安全阀，但 **默认应后置**，否则易在 A3a 未验证前引入归因噪声。

评审结论：

```text
025 可以立项；
建议有条件批准；
必须先过 Phase 0（B0 复现门）再开 A3a；
A3b / A3c 按阶段闸门解锁；
Phase 4 prod 提名默认禁止，除非探索 + 投产双轨 PASS 且项目经理书面批准。
```

### 3.2 缺口来源分析 — **通过，但需量化 A3a 上界**

025 §2.2 指出 phase1c `slow_up_open_count=0`、`watch_slow_uptrend` 大量 watch 无仓，是 coverage 最大 **潜在** 来源。与 023 §2 诊断一致（534 bar watch、0 开仓）。

需同时记入 **已覆盖风险**：

| 历史实验 | 改动 | test return | coverage | 教训 |
|----------|------|-------------|----------|------|
| 023 phase1a | 仅启用 `slow_up_position` + 0062e | **1.99%**（−7.32pp） | 26.67% | 慢涨无参与门 → 收益崩塌 |
| 024 A2 @ wp≥0.37 | TEQ 阈值放松 | 8.55% | 30.0% | 多 1 次 teq → 边际腿质量差 |
| 024 A2 @ wp=0.35 | TEQ 最佳 | **9.01%** | 26.7% | 当前 B0 |

025 假设：「趋势已确认、规则允许、legacy edge 未过线」的 bar 可通过 `participate_score` / 通道专用 edge 增量覆盖。**逻辑合理**，但 A3a 需在 **B0 已 26.7%** 基线上再抬 1.3pp——新增 slow_up 开仓必须与既有 teq 覆盖 **低重叠**，否则 coverage 分子不增或增量极小。

**修订建议（H1）**：Phase 1 交付 `slow_up_incremental_coverage_pp`（相对 B0 的 leg 增量覆盖，非绝对值），作为 A3a 是否继续的 **首要判据**；若增量 <0.8pp 且 return −>1pp，按 §9 关闭 A3a。

**评审意见：有条件通过。**

### 3.3 探索线 vs 投产线 — **通过方向，需降低 Phase 4 预期**

| 轨道 | return | coverage |
