# Document Archive

`document/` 按版本号归档。每次文档变更归入对应版本目录，避免同级堆积。

## 目录规则

- `001/`：v001 相关文档
- `002/`：v002 相关文档
- `003/`：v003 相关文档
- `004/`：v004 相关文档
- `005/`：v005 相关文档
- `006/`：v006 相关文档
- `007/`：v007 相关文档
- `008/`：v008 相关文档（0062 修复轮；含 `项目进度纪要.md`）

每轮文档变更完成后，在对应版本目录下补充 **项目进度纪要**，记录架构师终审结论与投产裁定。

后续新增文档时，请直接放入对应版本目录；若是新一轮变更，新增下一个版本目录。
