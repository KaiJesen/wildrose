```text
v1.1 阻塞项修订完成 → 可进入 Phase 0（B0 复现 + C3）
Phase 0 出口：part_auc ≥ 0.60 + B0 PASS + cum_IC 退化 ≤ 8%

Phase 3 入口：模型轨 part_auc ≥ 0.62 且 cum_IC ≤ 10%（双轨）
Phase 3 出口：探索双门 PASS（return ≥ 8.84% 且 coverage ≥ 28%）
Phase 4 出口：投产双门 PASS 方可提名 prod checkpoint
```
