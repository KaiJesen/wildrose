
def 上轨道寻找规则：
    假设上一个区域的最高点序号是a，当前的序号是t，[a,t]之间的最低点位置为b。在[a,b]之间找最高价格的点对，满足[a,b]间的所有最高价格都在其下；可以找到一个候选点对的数组，选择其中斜率最接近0的则为上轨线。

def 下轨道寻找规则：
    假设上一个区域的最低点序号是a，当前的序号是t，[a,t]之间的最高点位置为b。在[a,b]之间找最低价格的点对，满足[a,b]间的所有最低价格都在其上；可以找到一个候选点对的数组，选择其中斜率最接近0的则为下轨线。

def 做多平仓线计算规则：
    假设上一个区域的最低点序号是a，当前序号是t，[a,t]之间的最高点位置为b。在[a,b]之间找最低价格的点对，满足[a,b]间的所有最低价格都在其上；可以找到一个候选点对的数组，选择其中斜率最远离0的则为做多平仓线。

def 做多空仓线计算规则：
    假设上一个区域的最高点序号是a，当前序号是t，[a,t]之间的最低点位置为b。在[a,b]之间找最高价格的点对，满足[a,b]间的所有最高价格都在其下；可以找到一个候选点对的数组，选择其中斜率最远离0的则为做空平仓线。

def 当前区域为**做多观测区**的处理规则：
    当前区域为**做多观测区**意味着当前趋势是下降的，我们的目标是在该区间的后续发展中找到反转的位置。做多观测区一定有**上轨道线**。
    持续观察实时K线，会有以下情况：
    1. 最新的价格没有穿过上轨道线，保持在上轨道线之下：这种情况说明还没有实现突破，继续观察即可；这种情况不会更新上轨道线；
    2. 最新的价格有下而上穿过上轨道线，这种情况会有2种衍生的情况：
        2.1. 下一个K柱的收盘价格依然在上轨道线之上：认为是一个真实的突破。关闭**做多观测区**，同时进入**做多持仓区**，按照**做多平仓线**的计算规则初始化**做多平仓线**；
        2.2. 下一个K柱的收盘价格回到了上轨道线之下：认为这是一个诱多，不做理会；
    3. 价格出现当前**做多观测区**的新低：需要更新上轨，具体参考**上轨寻找规则**；

def 当前区域为**做空观测区**的处理规则：
    当前区域为**做空观测区**意味着当前趋势是上行的，我们的目标是在该区间的后续发展中找到反转的位置。做空观测区一定有**下轨道线**。
    持续观察实时K线，会有以下情况：
    1. 最新的价格没有穿过下轨道线，保持在下轨道线之上：这种情况说明还没有实现突破，继续观察即可；这种情况不会更新下轨道线；
    2. 最新的价格有上而下穿过上轨道线，这种情况会有2种衍生的情况：
        2.1. 下一个K柱的收盘价格依然在下轨道线之下：认为是一个真实的突破。关闭**做空观测区**，同时进入**做空持仓区**，按照**做空平仓线**计算规则初始化**做空平仓线**；
        2.2. 下一个K柱的收盘价格回到了下轨道线之上：认为这是一个诱空，不做理会；
    3. 价格出现当前**做空观测区**的新高：需要更新下轨，具体规则参考**下轨寻找规则**；

def 当前区域为**做多持仓区**的处理规则：
    当前区域为**做多持仓区**表示上一个区域为**做多观测区**。
    if 当前的价格从上到下突破**做多平仓线**：
        平多，进入**做空观测区**，并且按照**上轨寻找规则**初始化上轨道线；
    else if 当前没有**做多平仓线** or 在区域内出现新的最高价：
        计算做多平仓线，参考**做多平仓线**计算规则；
    else:
        当前什么也没有发生，什么也不用做；

def 当前区域为**做空持仓区**的处理规则：
    当前区域为**做空持仓区**表示上一个区域为**做空观测区**。
    if 当前的价格从下到上突破**做空平仓线**：
        平空，进入**做多观测区**，并且按照**下轨寻找规则**初始化下轨道线；
    else if 当前没有**做空平仓线** or 在区域内出现新的最低价：
        计算做空平仓线，参考**做空平仓线**计算规则；
    else:
        当前什么也没有发生，什么也不用做；

def 第一个区域初始化规则：
    在正式运行状态机前，预留 warmup 根 K 线（建议 >= 40）。
    在 warmup 区间内找全局最高点 H（序号 i_H）与全局最低点 L（序号 i_L）。
    若 |close[t]-close[0]|/close[0] < 震荡阈值（如 2%），继续等待更多 K 线。
    若 i_H < i_L（先高后低，下跌结构）→ 进入**做多观测区**：
        a = i_H，t = 当前序号，b = [a,t] 间最低点序号，按**上轨道寻找规则**初始化上轨。
    若 i_L < i_H（先低后高，上涨结构）→ 进入**做空观测区**：
        a = i_L，t = 当前序号，b = [a,t] 间最高点序号，按**下轨道寻找规则**初始化下轨。
    初始化完成后，后续状态切换完全由观测区/持仓区规则驱动，不再重复初始化。
